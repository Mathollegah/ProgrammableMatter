# Configgen

## Purpose of this Module
This module is used to generate pesudorandom configurations or load configurations from files.

## How to use this Module
An object of the class is created with the ```config``` containing parameter like the file to load form, and the ```state``` to which the generated configuration is appended. When ````config.load_file`` is set, the configuration is loaded from the specified file. Otherwise, the configuration is newly generated by using one of the following methods:

1) ```not config.random_configuration```
In this case the condiguration is created by starting with a single tile and adding a new tile to a random position of the current configuration. This is continued until ```config.number_of_tiles``` tiles are in the configuration. Note that this generation method has a bias, since tile created earlier have probably more neighbors.

2) ```config.random_configuration```
In this case random tiles are added inside of the bounding box until a connected component of at least size ```config.number_of_tiles``` is created. This component is then used as configuration while the others are deleted.
