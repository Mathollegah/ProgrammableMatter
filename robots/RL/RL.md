# Reinforcement Learning

## Purpose of this Module
This module is used for reinforcement learning on the configurations. Further, it can be used simulate the trained model as player.

## How to use this Module
This module constains two classes. Both classes inherit from ```RLRobotBase```. The class ```RLRobotLearn``` is used to train a model on random configurations. A instance of this class takes the configuration and the state of the game.

The other class is ```RLRobotPlay``` which is used to play the model. A instance of this class also takes the configuration and the state.
